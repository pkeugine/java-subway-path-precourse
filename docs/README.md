# PKEUGINE 브랜치

## 🛠 기능 구현 목록

- [x] 화면 출력
    - [x] 메인 화면 출력
    - [x] 경로 기준 선택 화면 출력
- [x] 선택 유효성 확인
    - [x] 입력한 옵션이 정수인지 확인
    - [x] 입력한 옵션이 해당 메뉴에서 사용할 수 있는 것인지 확인
- [x] 사용자의 출발역 입력
- [x] 사용자의 도착역 입력
  
- [ ] 입력한 역이 유효한지 확인
    - [x] 존재하는 역인지 확인
    - [x] 출발역과 도착역이 동일하지 않은지 확인
    - [ ] 역이 서로 연결되어 있는지 확인
- [ ] 최단 거리 조회 결과
    - [ ] 이동한 역 정보 출력
- [ ] 최소 시간 조회 결과
    - [ ] 이동한 역 정보 출력
  
- [x] 초기 역 / 노선 정보 추가
- [ ] 초기 역 / 노선간 거리와 시간 추가
